Aspect Z-Score T-Score
When to Use • Large sample (n ≥ 30)
• Population standard deviation (σ) known
• Population normally distributed 		• Small sample (n < 30)
• Population standard deviation (σ) unknown
• Sample standard deviation (s) used
• Population normally distributed
Independent Variable (X) • The manipulated/predictor variable
• What you control/change
• Examples:
- Treatment vs Control
- Before vs After
- Different methods/techniques 		• Same as Z-score
• Categorical (groups)
• Usually nominal or ordinal
• Examples:
- Treatment groups
- Time points
- Methods compared
Dependent Variable (Y) • The outcome/response variable
• What you measure
• Must be continuous/numerical
• Examples:
- Test scores
- Measurements
- Counts 		• Same as Z-score
• Must be continuous/numerical
• What you're testing for differences
• Examples:
- Performance measures
- Physical measurements
- Rates or scores
Basic Formula z = (x - μ)/σ t = (x̄ - μ)/(s/√n)
One Sample Formula z = (x̄ - μ)/(σ/√n) t = (x̄ - μ)/(s/√n)
Two Sample Formula z = (x̄₁ - x̄₂)/√(σ₁²/n₁ + σ₂²/n₂) t = (x̄₁ - x̄₂)/√(s₁²/n₁ + s₂²/n₂)
Critical Values (α = 0.05) • Two-tailed: ±1.96
• One-tailed: ±1.645 		Varies with df:
• df=1: ±12.706
• df=10: ±2.228
• df=30: ±2.042
• df=∞: ±1.96 (same as z)
Variable Examples Independent (X):
• Teaching methods
• Drug treatments
• Age groups

Dependent (Y):
• Test scores
• Blood pressure
• Recovery time 		Independent (X):
• Study techniques
• Exercise programs
• Teaching styles

Dependent (Y):
• Learning outcomes
• Weight loss
• Student performance
Note:
μ = population mean
x̄ = sample mean
σ = population standard deviation
s = sample standard deviation
n = sample size
df = degrees of freedom
X = Independent variable
Y = Dependent variable
