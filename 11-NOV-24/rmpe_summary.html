1. Basic Concepts

Hypothesis Testing

Null Hypothesis (H₀)
  • States no significant difference exists
  • Contains "equal to" (=)
  • Default position to be tested
Alternative Hypothesis (H₁)
  • States a difference exists
  • Opposite of H₀
  • What we're trying to prove
Common Hypothesis Forms: H₀: μ₁ = μ₂ (two groups equal) H₁: μ₁ ≠ μ₂ (two groups different) Or: H₀: μ = μ₀ (sample equals population) H₁: μ ≠ μ₀ (sample differs from population)

2. T-Test Analysis

Purpose

  • Compares means of TWO groups
  • Shows direction of difference
  • Uses t-distribution

Formula

One Sample: t = (x̄ - μ) / (s/√n) Two Sample: t = (x̄₁ - x̄₂) / √(s²₁/n₁ + s²₂/n₂) Where: x̄ = sample mean μ = population mean s = standard deviation n = sample size

Rejection Regions

Two-Tailed Test (α = 0.05) Reject H₀ Accept H₀ Reject H₀ ←───────|─────────────────|───────→ -2.01 +2.01

3. F-Test Analysis

Purpose

  • Compares variance between 2+ groups
  • Shows if differences exist (not direction)
  • Uses F-distribution

Formula

F = MSB / MSW Where: MSB = Mean Square Between groups MSW = Mean Square Within groups

F-Distribution

F-Test (α = 0.05) Accept H₀ | Reject H₀ ←────────────────|─────────────────→ F-critical

4. Test Comparisons

Aspect T-Test F-Test
Groups Two only Two or more
Direction Shows direction No direction
Distribution t-distribution F-distribution
Values Positive/Negative Only positive

5. Decision Making

T-Test Decision Tree

Calculate t-value ↓ |t| > critical value? ↙ ↘ Yes No ↓ ↓ Reject H₀ Accept H₀ ↓ Check direction

F-Test Decision Tree

Calculate F-value ↓ F > F-critical? ↙ ↘ Yes No ↓ ↓ Reject H₀ Accept H₀ ↓ Groups differ

6. Example Problems

T-Test Example

Problem: Compare two painkillers Data: Painkiller A: n=10, x̄=3.5, s=1.2 Painkiller B: n=10, x̄=5.8, s=1.4 t = (3.5 - 5.8) / √(1.2²/10 + 1.4²/10) t = -3.945 |t| = 3.945 > 2.01 (critical value) Decision: Reject H₀ Conclusion: Painkiller A more effective

F-Test Example

Problem: Compare three teaching methods Data: Method A: mean=85, SD=4, n=30 Method B: mean=78, SD=5, n=30 Method C: mean=82, SD=3, n=30 Calculate: 1. MSB = 245.3 2. MSW = 18.7 F = 245.3/18.7 = 13.12 F > F-critical (3.101) Decision: Reject H₀ Conclusion: Methods differ significantly

7. Quick Reference

Critical Values

Test Type α = 0.05 α = 0.01
Two-tailed t-test (df=30) ±2.042 ±2.750
One-tailed t-test (df=30) ±1.697 ±2.457
F-test (df1=2, df2=30) 3.32 5.39

Important Notes

  • Always state H₀ and H₁ clearly
  • Check assumptions before testing
  • Consider practical significance
  • Use appropriate significance level
  • Report both statistical and practical conclusions
