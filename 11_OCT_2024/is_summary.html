Hypothesis Testing - Complete Summary

1. Types of Hypothesis Tests

Test Type Null Hypothesis (H₀) Alternative (H₁) Usage
Two-Tailed μ = μ₀ μ ≠ μ₀ Testing for any difference
Left-Tailed μ ≥ μ₀ μ < μ₀ Testing for decrease
Right-Tailed μ ≤ μ₀ μ > μ₀ Testing for increase

2. Hypothesis Testing Process

  1. State the null and alternative hypotheses
  2. Choose significance level (α)
  3. Calculate test statistic
  4. Find p-value or critical value
  5. Make decision and state conclusion

3. Important Formulas

Z-Test Statistic:

Z = (x̄ - μ₀)/(σ/√n)
Use when: Population standard deviation known

T-Test Statistic:

t = (x̄ - μ₀)/(s/√n)
Use when: Population standard deviation unknown

Standard Error:

SE = s/√n
s = sample standard deviation, n = sample size

4. Decision Making Criteria

Using p-value Using Critical Value
If p < α: Reject H₀
If p ≥ α: Fail to reject H₀ 		If |test statistic| > critical value: Reject H₀
If |test statistic| ≤ critical value: Fail to reject H₀

5. Common Test Statistics

Test When to Use Key Assumptions
Z-test Known σ, large sample Normal distribution or n>30
T-test Unknown σ Normal distribution
Paired t-test Before/after measurements Normal distribution of differences

7. Important Considerations

  • Never "accept" the null hypothesis - only "fail to reject"
  • Statistical significance ≠ Practical significance
  • Sample size affects power of the test
  • Always check assumptions before testing
  • Consider effect size along with p-value
  • Larger sample sizes increase test precision

8. Common Mistakes to Avoid

  • Using one-tailed test without justification
  • Ignoring test assumptions
  • Confusing statistical and practical significance
  • Drawing causal conclusions from correlation
  • Ignoring the context of the problem
